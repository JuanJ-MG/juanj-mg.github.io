## Overview

This paper presents the **design, implementation, and validation** of a **CANSAT prototype** aimed at detecting vegetation areas using **aerial imaging**. The system integrates sensing, onboard processing, and data analysis to support **precision agriculture** applications.

---

## Focus

- CANSAT system architecture and integration  
- Image-based vegetation detection  
- Data acquisition and processing for agricultural monitoring
